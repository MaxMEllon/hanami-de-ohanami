### /users/sign_up => Web::users#create

#### request body

```
{
  user: [Schema] {
    email: [String] サインイン時に利用するメールアドレス
    password: [String] サインイン時に利用するパスワード
    password_confirmation: [String] パスワード確認用
  }
}
```

#### response body

```
{
  token: [String] JSON web token
}
```
